## 🔧 브라우저 설정

### Chrome (추천) ✅
- 음성 인식 완벽 지원
- 한국어/영어 모두 가능

### Safari ❌
- Web Speech API 미지원
- 텍스트 입력만 가능

### 마이크 권한 설정

**첫 실행 시:**
1. 🎤 버튼 클릭
2. "허용" 클릭 (브라우저 상단)
3. 다시 🎤 버튼 클릭

**권한 거부했다면:**
- Chrome 설정 → 개인정보 및 보안 → 사이트 설정 → 마이크
- localhost:8800 찾아서 "허용"으로 변경

## 🎯 기술 스펙

### 음성 인식
- **엔진**: Web Speech API (Chrome)
- **언어**: 한국어 (ko-KR)
- **정확도**: ~90%+ (명확한 발음)

### 처리 흐름
```
음성 🎤
  ↓
텍스트 변환 (Web Speech API)
  ↓
LLM 코드 생성 (GPT-4)
  ↓
로봇 실행 (MuJoCo)
```

## 🐛 문제 해결

### "음성이 감지되지 않았습니다"
- 마이크가 음소거되지 않았는지 확인
- 시스템 설정에서 마이크 권한 확인
- 다른 앱이 마이크 사용 중인지 확인

### "브라우저가 음성 인식을 지원하지 않습니다"
- Chrome 브라우저로 변경
- 최신 버전으로 업데이트

### 음성은 인식되는데 명령 실행 안 됨
- 터미널 로그 확인
- OpenAI API 키 설정 확인
- 서버 상태 확인
