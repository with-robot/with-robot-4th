## 📋 데모 명령어 (복사해서 사용)

### 기본 패턴
```
정사각형으로 움직여줘
원형으로 천천히 돌아줘
삼각형을 그려줘
제자리에서 한 바퀴 회전해줘
```

### 커스터마이징
```
2미터 크기로 정사각형 그려줘
반지름 1미터로 원형으로 돌아줘
큰 삼각형을 그린 다음 원점으로 복귀해줘
```

### 복잡한 시퀀스
```
정사각형을 그린 다음 원형으로 돌아줘
삼각형을 그리고 제자리에서 2바퀴 회전해줘
```

### 영어
```
Move in a square pattern
Draw a circle and return home
```

## 🔧 긴급 문제 해결

### API 키 오류
```bash
export OPENAI_API_KEY="sk-proj-..."
```

### 포트 충돌
```python

```

### 서버 재시작
```bash
# Ctrl+C로 종료 후
python llm_main.py
```
